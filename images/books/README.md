# כריכות ספרים

כדי שתמונות הכריכות יופיעו באתר, יש לשים כאן קבצי תמונה עם השמות הבאים:

| קובץ | ספר |
|------|-----|
| `certificat-lechayim.jpg` | סרטיפיקט לחיים |
| `zel-haazderech.jpg` | צל האזדרכת |
| `lirot-besdot-zarim.jpg` | לראות בשדות זרים |
| `aronit-kesuma.jpg` | הארונית הקסומה של סבתא |
| `penicilin.jpg` | פניצילין ועוד סיפורים |

**אפשרות 1:** להריץ פעם אחת (מחשב עם Node):
```bash
node scripts/download-book-covers.js
```
הסקריפט מוריד את הכריכות מהמקור המקורי ושומר כאן עם השמות הנכונים.

**אפשרות 2:** להוריד את התמונות ידנית מהאתר/מוציא לאור ולשמור עם השמות הרשומים למעלה.
